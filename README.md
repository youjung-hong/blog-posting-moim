# 블로그 글쓰기 모임


## 취지

- 잊어버리는 것이 너무 많다. 그러면 정리해서 추후 찾는 시간이라도 줄이자!
- 글쓰는 습관 들이기
- ~~죽어가는 블로그 소생시키기~~

------

## 규칙

​	이정돈 지키자구요~ 🧑‍💻

1. 2주일에 1개 포스팅 하기
   - 주제 : 자유 (책 정리를 하든, TIL을 하든, 일하다 모르는 내용을 정리든 상관 ❌ 날먹만 🙅‍♂️)
   - 마감 기한 : 그 주 일요일 자정까지
   - 글 길이 : readmin 7~8분 길이 였으면 해요?
2. 벌금
   - 1회에 만원
   - 모와서 카페 혹은 저녁을 먹어요~ 🍗🍽
3. 1달에 1회 포스팅
   - 자유
   - 마감 기한 : 그 달의 말일 자정까지
   - 글 길이 : 이 포스팅은 글 길이에 대한 제한을 두지 않아요~
